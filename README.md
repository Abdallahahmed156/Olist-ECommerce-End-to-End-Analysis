## 📌 Project Overview
This project provides a comprehensive data analysis of the **Brazilian E-Commerce Public Dataset by Olist**. The goal was to transform raw transactional data into actionable business insights, focusing on logistics performance, pricing strategies, and customer behavior.

## 🚀 Key Features & Workflow
- **Data Cleaning & Integration:** Merged 6 different datasets (Orders, Payments, Reviews, etc.) and handled missing values using logical business imputation.
- **Outlier Detection:** Applied the **Interquartile Range (IQR)** method to identify extreme pricing and flag "Business Anomalies" (e.g., shipping costs exceeding product prices).
- **Feature Engineering:** Created new metrics like `Delivery Lead Time`, `Arrival Accuracy`, and `Purchase Seasonality`.
- **Interactive EDA:** Developed dynamic visualizations using **Plotly** to explore shopping peaks and geographic distribution.

## 📊 Key Business Insights
1. **Logistics:** 85% of orders are delivered before the estimated deadline, showcasing high operational efficiency.
2. **Geography:** **São Paulo (SP)** is the dominant market, contributing to over 40% of total sales.
3. **Consumer Behavior:** Peak shopping activity occurs between **10:00 AM and 4:00 PM**, primarily on weekdays.

## 🛠️ Built With
- **Python:** For data processing and engineering.
- **Plotly:** For interactive storytelling.
- **Pandas/NumPy:** For statistical analysis.
